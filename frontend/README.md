# Nutrionista Frontend

## Paigaldus

```bash
npm install
```

## Käivitamine

```bash
npm run dev
```

Ava brauseris: http://localhost:5173/

## Build

```bash
npm run build
```

## Nõuded

- Node.js 18+ (soovituslik 20+)
- npm

## Projekti struktuur

- `src/views/` - lehevaated
- `src/components/` - taaskasutatavad komponendid
- `src/router/` - Vue Router seadistus
- `src/services/` - API teenused
